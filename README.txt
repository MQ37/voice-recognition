Voice recognition
=================

 Semestral project: voice recognition utilizing signal processing techniques.

Assignment
==========

 Find spoken words in the audio signal, which can only contain the words listed below.
 Words: time, prepare, solution, make, mistake, no, the, probable, long, lecture, method, disaster, fail, work, advice, idea, succeed, easy, is, for, give.

 Use just conventional signal processing techniques, using machine learning is not allowed.
 Files:
    * Signal1.txt - single words split by a pause
    * Signal2.txt - testing data (without a pause)
    * Signal3.txt - testing data (without a pause)

Notes
=====

 NumPy loadtxt function can be used to load the audio signal from the files.
